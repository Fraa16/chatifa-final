## How to go live on Vercel

1. Go to https://vercel.com
2. Create a free account
3. Click "Add New Project"
4. Look for "Upload" or drag this entire folder into the browser
5. Click Deploy
6. Vercel gives you a live URL (e.g. chatifa.vercel.app) within 60 seconds

## How to add your domain (chatifa.de)

1. In Vercel dashboard → your project → Settings → Domains
2. Type in "chatifa.de" and click Add
3. Vercel shows you DNS settings — log into your domain registrar
   and add those records (usually 2 lines: A record + CNAME)
4. Wait 5–30 minutes → site is live at chatifa.de
